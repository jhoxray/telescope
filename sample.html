Welcome to the Telescope Logger sample app

This is a simple demo app to illustrate capabilities of the Telescope Logger package that provides simple logging for the Meteor framework. The actual example of how it looks is right below. Logs are sortable, just click on the column title.
Why?
I got frustrated with the need to switch between the terminal and Chrome's js console to keep track of Meteor app logs, plus needed a basic standard logging package that can support different log levels and present all that in one place in a nice format.
What does it do?
Supports different log levels (currently FATAL, ERROR, WARNING, INFO, VERBOSE) with corresponding methods for message output, optional logging to console, pretty output of both Client and Server logs using Twitter Bootstrap right in the browser.
Usage

For details, see Telescope Logger package on GitHub (it contains this app and Telescope Logger library), but in a nutshell you simply call
TL=new TLog(TLog.LOGLEVEL_MAX,true) once in your app's common code (that gets executed on both client and server) and then call the following obvious methods: TL.fatal("..."), error, warn, info, verbose.

To display the logs, just plugin logs_bootstrap Handlebars template anywhere in your html pages. E.g., in this app it's in the column to the right.

{{>logs_bootstrap}}
